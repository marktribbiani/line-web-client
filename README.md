# README

Line web client is a forked version of Line Chrome Extension. You can use this client to sign in with multiple Line accounts. You can use the online version [https://www.linewebapp.com/](https://www.linewebapp.com/) or even deploy your own Line web client.

## Deploy your own

Deploy your own Line web client using [Vercel](https://vercel.com) or preview live with [https://www.linewebapp.com/](https://www.linewebapp.com/)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fmarktribbiani%2Fline-web-client)
